Derogatory comment Classification

We interfaced with the PMU provided by modern CPUs to sample the occurrences of various events in the system. A linux kernel module was written which calculates the amount of time spent in each instruction function of a program; and the number of L1 caches misses, branch mispredictions at instruction level. A comparison with the the tool perf was done to validate the results.

Global variable accesses made in procedures

An access based interprocedural analysis was done to partition the data space for procedures based on the variables accessed within a procedure and its callees. We discovered access of variables locally and propagated information over the call graph. Implemented this module as a plugin in gcc-4.7.2 by analyzing GIMPLE, the intermediate representation (IR) of GCC.

Modeled and Animated a Transformer using OpenGL

We created an interactive transformer model using OpenGL primitives. A basic animation recording interface to capture keyframes was implemented. Stock Price Prediction

In my stock price prediction project, I meticulously extracted historical market data from reliable sources such as Yahoo Finance and Alpha Vantage, focusing on key indicators like opening prices and trading volumes. Through thoughtful feature engineering and careful data cleaning, I crafted a robust dataset for the application of LSTM and SVM algorithms. The models exhibited strong predictive accuracy, with comparative analysis showcasing the versatility of LSTM in capturing intricate patterns and SVM's effectiveness in handling complex market relationships. This project not only harnessed advanced machine learning techniques but also adhered to ethical considerations, showcasing a comprehensive approach to stock price prediction.

An accuracy of 90% was achieved.
